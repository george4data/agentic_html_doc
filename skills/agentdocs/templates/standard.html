Overview

Overview

Tokens are signed with RS256 and expire after 15 minutes. The client sends the token as a Bearer credential on every authenticated request.

API

POST /auth/login

Token flow

Canonical example

const res = await fetch("/auth/login", {
  method: "POST",
  body: JSON.stringify({ email, password }),
});
const { token } = await res.json(); // 15-min RS256 JWT

This example is shown statically and pinned to a tested file; its trust comes from the pin, not from running anything here.
